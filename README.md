# NoveFilmes
Nove Filmes é uma empresa de locação de equipamentos de audiovisual e fotografia

## Como visualizar (até o momento):
1. abra um terminal no diretório que contém os arquivos: README.md, package.json,tsconfig.json, etc...
2. digite o comando `npm run dev`
3. abra o url local exibido no terminal (Exemplo: https://localhost:3000).
